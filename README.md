# Proyecto DevOps - Sistemas Operativos II

## Descripcion

Este proyecto consiste en la implementacion de una infraestructura DevOps en la nube, utilizando Docker, GitHub, GitHub Actions y AWS.

## Objetivo

Diseñar, implementar y documentar una aplicacion contenerizada con despliegue en la nube y automatizacion CI/CD.

## Tecnologias utilizadas

- Node.js
- Express
- Docker
- Docker Compose
- GitHub
- GitHub Actions
- AWS
- Ubuntu Server

## Endpoints

- `/` Pagina principal
- `/health` Estado de la aplicacion
- `/api/tareas` API de tareas del proyecto
